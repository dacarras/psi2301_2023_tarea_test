# funcin para obtener moda
mode <- function(v) {
   uniqv <- unique(v)
   uniqv[which.max(tabulate(match(v, uniqv)))]
# source: https://www.tutorialspoint.com/r/r_mean_median_mode.htm
}


# tabla descriptiva

Resumen

  • La distribucin de ingresos de los hogares es muy asimetrica.
  • Presenta una asimetra positiva, donde los ingresos ms altos se concentran en muy pocos hogares.
  • Debido a que la distribucion de ingresos es muy asimtrica, la media de la distribucin es poco informativa.
  • La mediana de ingresos es de 307.6292917 mil pesos.
  • Un 75% de los hogares presenta ingresos menores a 580 mil pesos mensuales.
